# Python para Data Science

## Descrição

Este módulo cobre o uso de Python para análise e ciência de dados, incluindo:

- Fundamentos de Python
- NumPy e Pandas para manipulação de dados
- Matplotlib e Seaborn para visualização
- Análise exploratória de dados (EDA)
- Machine Learning com Scikit-learn
- Estatística aplicada

## Estrutura

- `/exercicios`: Exercícios práticos e projetos de análise de dados

## Materiais de Estudo

Os conteúdos e exercícios serão organizados por tópicos para facilitar o aprendizado progressivo.
