# Banco de Dados

## Descrição

Este módulo aborda os fundamentos e práticas de bancos de dados, incluindo:

- Modelagem de dados
- SQL e consultas avançadas
- Bancos de dados relacionais (MySQL, PostgreSQL)
- Bancos de dados NoSQL (MongoDB, Redis)
- Normalização e otimização
- Transações e integridade de dados

## Estrutura

- `/exercicios`: Exercícios práticos e projetos

## Materiais de Estudo

Os conteúdos e exercícios serão organizados por tópicos para facilitar o aprendizado progressivo.
