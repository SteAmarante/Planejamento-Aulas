# Planejamento de Aulas

## Descrição

Este repositório contém materiais de estudo organizados para diferentes disciplinas de tecnologia e dados. O conteúdo está estruturado em módulos temáticos, cada um com exercícios práticos para consolidação do aprendizado.

### Módulos Disponíveis

- **Banco de Dados**: Conceitos, modelagem e prática com bancos de dados relacionais e não-relacionais
- **Python para Data Science**: Programação em Python aplicada à análise e ciência de dados
- **Power BI**: Desenvolvimento de dashboards e visualizações de dados
- **IA Generativa**: Introdução e aplicações práticas de inteligência artificial generativa
