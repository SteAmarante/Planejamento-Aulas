# Power BI

## Descrição

Este módulo ensina a criar dashboards e relatórios interativos com Power BI, incluindo:

- Introdução ao Power BI Desktop
- Conexão e transformação de dados (Power Query)
- Modelagem de dados e relacionamentos
- Medidas DAX e cálculos
- Visualizações e dashboards interativos
- Power BI Service e compartilhamento

## Estrutura

- `/exercicios`: Exercícios práticos e projetos de dashboards

## Materiais de Estudo

Os conteúdos e exercícios serão organizados por tópicos para facilitar o aprendizado progressivo.
