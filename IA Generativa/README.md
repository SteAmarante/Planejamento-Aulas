# IA Generativa

## Descrição

Este módulo explora o mundo da inteligência artificial generativa, incluindo:

- Fundamentos de IA e Machine Learning
- Modelos de linguagem (LLMs)
- Prompt Engineering
- APIs de IA (OpenAI, Anthropic, etc.)
- Aplicações práticas de IA generativa
- Ética e responsabilidade no uso de IA

## Estrutura

- `/exercicios`: Exercícios práticos e projetos com IA generativa

## Materiais de Estudo

Os conteúdos e exercícios serão organizados por tópicos para facilitar o aprendizado progressivo.
